# Screenshots 目录

这个目录用于存放项目预览截图。

## 建议的截图列表

请准备以下截图并保存到此目录：

- `home.png` - 主页/小说列表页面
- `novel-detail.png` - 小说详情页（大纲/章节/人物卡等）
- `chapter-create.png` - 章节创作页面
- `realtime-progress.png` - 实时生成进度页面

## 截图要求

- 格式：PNG 或 JPG
- 分辨率：建议 1920x1080 或更高
- 文件大小：尽量压缩，建议每张不超过 500KB

## 截图工具推荐

- **macOS**: Command + Shift + 4
- **Windows**: Win + Shift + S
- **在线压缩**: [TinyPNG](https://tinypng.com)

## 使用方法

截图准备好后，它们会自动在 README.md 的"项目预览"折叠部分显示。
